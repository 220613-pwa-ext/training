# XPath exercise #1

Exercise link: [https://chercher.tech/practice/dynamic-table](https://chercher.tech/practice/dynamic-table)

Write XPath that will select
* The checkbox next to Chercher.tech
* The checkbox next to facebook.com

**Note: The 2 XPaths should be written in such a way that they will work even after reloading the page. This is important because the order of the data in the table is NOT the same after each page reload!**
